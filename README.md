Resume Builder - AI Powered ATS Resume Generator

A modern full-stack Resume Builder that helps users create professional, ATS-friendly resumes with AI-assisted content generation. The application provides an intuitive interface for building resumes, generating professional summaries and work experience descriptions using Gemini AI, and exporting polished resumes for job applications.

Installation
Clone Repository
git clone https://github.com/your-username/resume-builder.git
cd resume-builder
Backend Setup
cd server

Install dependencies

npm install

Create a .env file

PORT=3000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY

GEMINI_API_KEY=YOUR_GEMINI_API_KEY

Run backend

npm run dev
Frontend Setup
cd client

Install dependencies

npm install

Create .env

VITE_BASE_URL=http://localhost:3000

Run frontend

npm run dev
Production Environment

Frontend:
